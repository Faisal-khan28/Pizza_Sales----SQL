# Pizza Sales Analysis using SQL

This project analyzes pizza sales data using SQL to uncover business insights such as revenue trends, popular pizza types, and ordering patterns.

## Project Objective

The objective of this project is to analyze pizza sales data using SQL in order to understand customer ordering behavior and identify key business insights such as:

- Total revenue generated from pizza sales
- Most popular pizza types and sizes
- Sales distribution by category
- Peak ordering hours
- Revenue contribution of different pizza categories

## Dataset

- Multiple CSV files located in pizza_sales_dataset
  - order_details
  - orders
  - pizza_types
  - pizzas

## Key Insights

- Total revenue generated is 817860.05
- Most common pizza size that is ordered is (L) with 18956 Orders
- Most ordered category is Classic and least ordered is Chicken
- Peak ordering hours are from 12PM to 1PM and 5PM to 7PM
- Revenue in percentage of each pizza category
  - Chicken - 23.96%
  - Supreme - 25.46%
  - Classic - 26.91%
  - Veggie - 23.68%
